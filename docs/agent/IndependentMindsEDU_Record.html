Independent Minds EDU

Project Record Document - Generated 2026-07-12

1. Project Purpose

Independent Minds EDU is a family-focused education platform designed to help parents and guardians manage homeschool workflows, monitor student progress, and support learning through planning tools, reporting, and AI-assisted tutoring.

2. Technical Foundation

  • Frontend: React + TypeScript + Vite + Tailwind + shadcn/ui
  • Backend: Supabase (Auth, Postgres, Edge Functions, Storage)
  • Hosting: Vercel
  • Domain: https://www.independentmindsedu.org
  • Notifications: Email, Telegram, WhatsApp/Twilio

3. Recent Stabilization Outcomes

  • Legacy Lovable-facing dependencies and references were migrated out of core flows.
  • Google OAuth callback handling was hardened for both query-code and hash-token return patterns.
  • Canonical host handling for login flow was improved to reduce apex/www mismatch issues.
  • User-facing sender branding was standardized to Independent Minds EDU identity.
  • Global lint gate was unblocked for ongoing delivery continuity.
